# 🌦️ Weather App Project

A modern and responsive Weather App built using HTML, CSS, and JavaScript.

## 🚀 Features

- 🌍 Search weather by city
- 🌡️ Live temperature display
- 💧 Humidity information
- 🌬️ Wind speed details
- 📱 Responsive design
- 🎨 Modern UI with animations
- 👨‍💻 Multi-page portfolio style layout
- 🖼️ Gallery section
- ❓ FAQ section
- 📞 Contact page

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- OpenWeather API
